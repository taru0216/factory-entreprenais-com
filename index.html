出荷の仕組み

  • Factory は在庫 HTML の生産・品質チェックを担当する。 本番サーバーとは分離されており、承認前に公開されることはない。
  • 出荷は GitHub Actions が自動実行する。 Astro ビルド → GitHub Pages へのデプロイが コードマージをトリガーに走る。
  • 各カテゴリ（marche / foodre / cities）は 独立したサブドメインに出荷される。 Factory からは複数の出荷先へ並列配送が可能。
  • Factory 自体は常時公開されており、 事業者はここから自社サイトのプレビューを確認できる。
